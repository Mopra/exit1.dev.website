# Plan Change Discord Announcement Template

Use this for pricing changes, tier changes, grandfathering, plan migrations, usage limits, or paid plan announcements.

```md
# :yellow_heart: Update to the plans

First of all, thank you for supporting exit1.

The product keeps getting more serious, and I want the plans to reflect that without making things confusing.

## What’s changing

• **[Plan/change 1]** — [Short explanation]  
• **[Plan/change 2]** — [Short explanation]  
• **[Plan/change 3]** — [Short explanation]

## Existing users

If you’re currently on **[old plan]**, you will be moved to **[new plan / grandfathered setup]**.

You keep **[important benefits they keep]**.

No action needed.

## Free users

Free is still available.

You still get:

• [Free benefit 1]  
• [Free benefit 2]  
• [Free benefit 3]

The main limit changing is:

• [Limit change, written clearly]

## Paid users

Paid plans now give you more room for serious monitoring:

• Faster checks  
• More history  
• More alerts  
• More status pages  
• More advanced monitoring features

I know plan changes are never the most exciting update, but they make it possible to keep building exit1 properly.

Thanks again for being here early.

As always, feedback is welcome.
```

## Notes

Best used when:

• You are changing pricing or limits  
• You need to reassure existing users  
• You are grandfathering users into better terms  
• You want to explain the business reason without sounding defensive

Be extra clear in this format. Pricing confusion creates more support work than almost anything else.
